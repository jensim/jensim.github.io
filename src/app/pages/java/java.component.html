mega-manipulator
My latest addition to the "pile". It's currently in use, and development.
Aims to address both ends of batch changes in code. Making and Receiving.
Leverage this IntelliJ-plugin to search, update, pull request, approve, merge, comment, modify..
The list goes on.
Pull request fatigue is a word that came up a lot when I started working with batch changes, Renovate and SourceGraph Batch Changes.
The goal is then, addressing the receiving side of batch changes, to make it as easy to receive the changes as it is to generate them.
